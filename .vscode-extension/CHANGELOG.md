# Change Log

All notable changes to the "SUB Language Support" extension will be documented in this file.

## [1.1.0] - 2026-07-09

### Added
- Code snippets for common SUB constructs (var, func, if, for, while, etc.)
- Smart indentation rules for SUB syntax
- Code folding support for functions, loops, and conditionals

### Improved
- Enhanced syntax highlighting for embedded code blocks
- Better bracket matching and auto-closing pairs

## [1.0.0] - Initial Release

### Added
- Syntax highlighting for SUB language
- Support for `.sb` and `.sub` file extensions
- Comment toggling support
- Basic language configuration
